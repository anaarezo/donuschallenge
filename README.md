## Installation


### Step 1
ðŸ‘¯ Clone this repo to your local machine using:
 `https://github.com/anaarezo/donuschallenge.git`


### Step 2
Open the terminal and access the project folder `./donuschallenge`.

### Step 3
Run `yarn` or `npm` to install the required packages (I recommend `yarn`)

For Yarn users: 
```shell
$ yarn
```
For NPM users:

```shell
$ npm install
```

### Step 4

Run `expo` and scan the QRCode at the screen

Yarn user:
```shell
$ yarn start
```
Npm user:
```shell
$ npm start
```
if none works try:
```shell
$ yarn expo start
```
